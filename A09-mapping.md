A09 mapping (summary)

This file documents how the portfolio website maps to the A09 marking scheme.

Presentation (A09-A) — 30%
- Use the `presentation` section as source material for the elevator pitch.
- Include a downloadable PowerPoint named according to the spec for submission.

Portfolio submission (A09-B) — 70%
- Professionalism, organisation and usability (20%): site layout, CV link, clear section ordering (About, Projects, Reflection, Submission).
- Evidence of technical and academic skills (30%): `projects` section lists curated examples (ICMG24 presentation; departmental digitising; NASA Space Apps; book chapter).
- Quality and relevance of showcased work (25%): projects contain short descriptions and emphasis on contributions and outcomes.
- Reflection and professional communication (15%): `reflection` section uses the spec template and contains brief, specific prompts to complete.
- LinkedIn & completeness (10%): prompt in About to add LinkedIn URL and ensure links function.

Print / PDF
- Use browser Print → Save as PDF to create the PDF submission. The site includes `@media print` rules to remove navigation and map for clean PDF export.

Notes / next steps
- Replace placeholder project descriptions with screenshots, maps or links to repositories for stronger evidence.
- Add a LinkedIn URL, presentation slides, and a finalized reflective statement (≤300 words).
- Optionally add screenshots and small thumbnails for each project, and verify external links before submission.
